# Home Assistant Add-On: ntfy.sh Server

[ntfy.sh](https://ntfy.sh) | Send push notifications to your phone or desktop via PUT/POST


## About

A [Home Assistant](https://home-assistant.io) add-on repository for a self hosted [ntfy.sh](https://ntfy.sh) Server.

[![Add Addon to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https://github.com/friendlydev/homeassistant-add-ons)

You will want to use this in conjunction with the HACS addon, which allows the use of a self hosted ntfy server.
[hbrennhaeuser/homeassistant_integration_ntfy](https://github.com/hbrennhaeuser/homeassistant_integration_ntfy)
